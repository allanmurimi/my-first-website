Why I Want To Learn Coding

    Passion

You have the idea, but now you need to depend on someone to build it for you. Knowing how to code, you can take matters in your hand and start bootstrapping one the same day. Save the trouble and build a working prototype yourself.

    Mind Engaging

The programming world is a fast-paced environment. It’s impossible to know everything, but it’s possible to specialize in a specific programming language or specific tooling and become really good at one particular programming stack. programming definitely keeps you busy as there’s a never-ending river of information to absorb.

    Fun Hobby

Not everyone has to work in a professional environment. Coding is much more than a corporate 8-5 job. Coding can be a fun hobby for anyone, there are no entry barriers. Anyone who has an internet connection and a computer can start coding as of this moment. Coding adheres to no limitations.
